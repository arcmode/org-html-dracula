> org-html-dracula

Dracula theme is a great choice for editors, UI, terminals, etc. Org-mode is a powerful tool for workflows involving text and code documents. This web site was exported by Org mode and the CSS code follows Dracula theme

3 How to use

Setup file

You can use this setup file which only adds a stylesheet at the moment of writing this 

#+SETUPFILE: https://drojas.github.io/org-html-dracula/setup.org

Stylesheet

Or you can include the stylesheet in your Org files directly 

#+HTML_HEAD: <link rel="stylesheet" type="text/css" href="https://drojas.github.io/org-html-dracula/styles/org.css"

3.1 Customized

This is actually the Org file that exports this HTML document looks like. 

#+TITLE: > org-html-dracula
#+AUTHOR: David Rojas
#+EMAIL: (concat "drojascamaggi" at-sign "gmail.com")

#+OPTIONS: toc:nil num:2 H:3 ^:nil pri:t title:nil
#+EXPORT_FILE_NAME: index

#+SETUPFILE: https://drojas.github.io/org-html-dracula/setup.org

#+BEGIN_export html
<h1 class="title">
  <span class="accent">></span> org-html-dracul<span class="inverse">a</span>
</h2>
#+END_export

#+BEGIN_abstract
Dracula theme is a great choice for editors, UI, terminals, etc. Org-mode is a powerful tool for workflows involving text and code documents.
This web site was exported by Org mode and the CSS code follows Dracula theme
#+END_abstract

#+BEGIN_QUOTE
"My old recipes look so cool now!"\\
-- Vlad III
#+END_QUOTE

#+BEGIN_export html
<span id="end-heading"></span>
#+END_export
