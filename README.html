> org-html-dracula

Dracula theme is my current default for code editing, gnome, terminal, etc. Org-mode is a great tool for workflows involving text and code documents that I'm starting to love. Hence I created this CSS stylesheet for Org HTML exports with Dracula colors!

3 How to use

This document was exported from this Org file. You can use the styles in any of the following manners

3.1 Simple

Stylesheet

The CSS source is here 

#+HTML_HEAD: <link rel="stylesheet" type="text/css" href="styles/org.css"/>

Setup file

The setup file only adds a stylesheet but may include other features in the future 

#+SETUPFILE: ./setup.org

3.2 Customized

The following snippet can be copied and pasted to start a new document. This is how the Org for of this document looks like.

Org HTML boilerplate

#+OPTIONS: toc:nil num:3 H:4 ^:nil pri:t title:nil
#+SETUPFILE: ./setup.org

# Document heading

#+begin_export html
<div id="floating-logo">
  images here will be seen at the top with "space between"
</div>
#+end_export

#+BEGIN_export html
<h1 class="title">
  <span class="accent">></span> org-html-dracul<span class="inverse">a</span>
</h2>
#+END_export

#+BEGIN_abstract
This text will appear in italics after the title
#+END_abstract

#+BEGIN_QUOTE
Quote you favorite fictional characters here :)
#+END_QUOTE

#+BEGIN_export html
<span id="end-heading"></span>
#+END_export

#+TOC: headlines 2
